### 第 1 步：打开 Netlify
点击这个链接：**https://app.netlify.com/drop**

### 第 2 步：拖拽文件夹
将整个项目文件夹拖拽到上传区域：
- 路径：`/Users/jiangli/Cursor/251031`

等待 10-30 秒，部署完成后您会获得一个网址！

---

## 详细步骤（带截图说明）

1. **访问 Netlify**
   - 网址：https://app.netlify.com
   - 如果没有账户，点击右上角 "Sign up" 注册（支持 GitHub 登录）

2. **选择部署方式**
   - 点击页面右上角的 **"Add new site"**
   - 在下拉菜单中选择 **"Deploy manually"**

3. **上传文件**
   - 在 Finder 中找到这个文件夹：`/Users/jiangli/Cursor/251031`
   - 将整个文件夹拖拽到 Netlify 的虚线框中
   - 等待上传和部署完成

4. **完成！**
   - 部署完成后，您会看到一个网址
   - 点击网址即可访问您的网站
   - 网站已经上线，全球可访问！

## 🎉 完成！

部署成功后，您的网站就可以通过 Netlify 提供的网址访问了。

如果需要自定义域名，可以在 Netlify 后台的 "Site settings" → "Domain management" 中设置。
