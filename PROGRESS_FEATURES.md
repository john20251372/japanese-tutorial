# 进度保存功能说明

## ✅ 当前已实现的功能

您的应用**已经**具备完整的进度保存功能！

### 自动保存
- ✅ **自动保存**：当您标记短语为"已掌握"或"未掌握"时，进度会自动保存到浏览器本地存储
- ✅ **持久化**：进度保存在浏览器的 localStorage 中，即使关闭浏览器或刷新页面，进度也不会丢失
- ✅ **跨会话**：下次打开网站时，您的所有进度都会自动恢复

### 保存的内容
1. **短语掌握状态**：所有分类中每个短语的掌握状态（已掌握/未掌握）
2. **单词掌握状态**：常用单词列表中每个单词的掌握状态
3. **用户偏好**：
   - 假名显示/隐藏状态
   - 中文显示/隐藏状态
   - 语音选择偏好

## 📊 如何查看进度

进度会显示在以下位置：
- **首页**：顶部进度条显示总体掌握情况
- **各分类卡片**：显示每个分类的掌握数量
- **学习/测试模式**：顶部显示当前分类的掌握进度

## 💾 进度存储位置

进度存储在浏览器的 **localStorage** 中，键名为 `japaneseProgress`。

### 查看存储的数据（开发者工具）

1. 打开浏览器开发者工具（F12 或 Cmd+Option+I）
2. 切换到 "Application"（或"存储"）标签
3. 在左侧选择 "Local Storage"
4. 点击您的网站域名
5. 查找键名 `japaneseProgress`，可以看到保存的所有进度数据

## 🔄 如果需要重置进度

如果您想清空所有进度，在浏览器控制台（Console）中输入：

```javascript
localStorage.removeItem('japaneseProgress');
location.reload();
```

## 💡 提示

- 进度是保存在**当前浏览器**中的
- 不同浏览器的进度是独立的
- 清除浏览器数据会删除进度
- 使用隐私/无痕模式时，关闭窗口会清除进度
